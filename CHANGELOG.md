# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0.0] - 2026-04-13

### Added
- Lamborghini-inspired design system across all pages (DESIGN.md)
- Full-viewport black hero section with giant uppercase 超跑百科 headline
- Gold CTA buttons (`#FFC000`) and transparent ghost buttons on all CTAs
- Charcoal card surfaces (`#181818`/`#202020`) with zero border-radius throughout
- Full-width cinematic hero image on car detail pages with gradient text overlay
- Gold-highlighted key specs (HP, torque, 0-100, top speed) on car detail page
- Uppercase badge system for year, series, and brand metadata
- Animated gold loading bar replacing spinner
- Lamborghini-style 404 page with 120px ghost numeral and gold CTA

### Changed
- Navigation redesigned: fully transparent, centered wordmark, floating nav links
- All page backgrounds switched to absolute black (`#000000`)
- Typography shifted to all-uppercase display text with tight line-heights
- Car grid cards now use surface-color hover (no JavaScript event handlers)
- Spec comparison table uses alternating dark surfaces instead of borders
- Search input styled with gold bottom border accent
- Compare page highlights winning spec values in gold
- Brand detail page shows faint cover image as atmospheric background

### Removed
- Rounded corners on all cards, buttons, and containers
- Semi-transparent backdrop-blur navigation bar
- Gold `#d4a853` legacy accent color replaced with Lamborghini Gold `#FFC000`
